## 累乗コマンド
標準入力から受け取った数字から、指定した数の累乗を計算し標準出力に表示するコマンドです。

## インストール方法
以下のコマンドを順に実行し、リポジトリをコピーしたいディレクトリでクローンして下さい。

```
$ https://github.com/taigerman0012/robosys2025.git
$ cd robosys2025
$ chmod +x report
```

## 使用方法
- 標準入力から[底] [最大乗数]の順でスペース区切りの数値を渡します。

```
$ echo "[底] [最大乗数]" | ./report
```

## 実行例
- 2の1乗から10乗までを表示する場合

```
$ echo "2 3" | ./report
2.0
4.0
8.0
```

- 1.5の5乗までを表示する場合

```
$ echo "1.5 5" | ./report
1.5
2.25
3.375
5.0625
7.59375
```
